Setting the runtime library path

To modify your run-time library loading path you may need to modify your existing system environment to ensure it contains the /usr/local/lib directory, assuming you used a default install perfix when you ran configure.

Linux

Linux. On linux, add the libsres and libval library location (E.G. /usr/local/lib) to /etc/ld.so.conf, then run ldconfig.
