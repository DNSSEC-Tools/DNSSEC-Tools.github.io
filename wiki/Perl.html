Perl

Perl is a extensible scripting language that people either love or hate. Many of the DNSSEC-Tools Components are written in perl since many supporting modules were used to base some of the software on. Perl is available from http://www.perl.org and http://www.activestate.com/ for windows. Almost every operating system comes with perl installed these days (except windows).
