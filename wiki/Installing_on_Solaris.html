Installing on Solaris

Mostly solaris users should follow the more generic Installing on Unix instructions, but with the following caveats:
  • You'll need either gcc or Sun's commercial C compiler
  • But, the default perl that comes with solaris has been compiled with Sun's commercial C compiler
    • compiling new C-based perl modules with gcc doesn't work with the default perl
    • the only solution is to either buy Sun's commercial compiler, or install a version of perl that is compiled with gcc.
