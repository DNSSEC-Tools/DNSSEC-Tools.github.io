ZSK

ZSKs are Zone Signing Keys which are keys used to sign all the RRsets in a zone. Key Signing Keys (KSK) differ from ZSKs as they only sign the key RRset contain in a zone file. See the KSK wiki node for more detailed information on both ZSKs and KSKs.
