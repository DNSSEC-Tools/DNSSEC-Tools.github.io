DNSSEC Capable Applications

Firefox

Firefox. A patched Firefox RPM is available as well as the patch itself. A patched Firefox will notify the user of DNSSEC validation failures.
    - firefox website - Jump to DNSSEC-Tools Software Summary

    Sendmail

    Sendmail. With the DNSSEC-Tools sendmail patch applied it will perform DNSSEC validating DNS queries when determining where to send mail. The mail will bounce back to the sender when an attempt is made to send mail to sites that do not pass DNSSEC validation. Once the patch is applied to a local sendmail installation, the only other configuration necessary (other than libval's dnsval.conf) is to add the 'RequireDNSSEC' option to ResolverOptions in sendmail's configuration.
      - sendmail website - Jump to DNSSEC-Tools Software Summary

      Postfix

      Postfix. Similar to the sendmail patch, once patched, postfix will perform DNSSEC validating DNS queries when determining where to send mail. The mail will bounce back to the sender when sent to sites that do not pass DNSSEC validation. Currently the postfix patch does not provide for configuring DNSSEC validation off and on.       LibSPF

      LibSPF. This patch adds DNSSEC validation to DNS queries in libspf2. There are patches for libspf2 versions 1.0.4 and 1.2.5.. Recompiling with this patch allows applications using libspf2 to also check for DNSSEC validation.
        - libspf2 website - Jump to DNSSEC-Tools Software Summary

      Thunderbird

      Thunderbird. This patch adds DNSSEC validation to the Thunderbird mail client. It requires getting, patching and building the Thunderbird source code. It has been tested with version 1.5.0.10 and should be considered beta.
        - thunderbird website - Jump to DNSSEC-Tools Software Summary

      proftpd

      proftpd. This patch adds DNSSEC validation using libval to proftpd. It has been tested against versions 1.3.0a and 1.3.1rc2 of proftpd. Using it requires patching and building a local version of proftpd.
        - proftpd website - Jump to DNSSEC-Tools Software Summary
