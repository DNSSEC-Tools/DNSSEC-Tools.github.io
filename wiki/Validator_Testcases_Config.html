Validator Testcases Config

This page specified the format of the configuration file used for validator test cases.

Overview

Overview. The validator application can run series of tests to exercise the library code for a variety of test cases. Originally these test cases were hardcoded.

Specification

Requirements

    Specification. Requirements.
  • Each test case should have a text description
  • Each test case should be a query for a single domain/class/type tuple
  • Each test case should have a specific set of expected validation results
  • Tests cases should be able to be grouped into a "suite" of tests
  • All tests for a suite must be specified sequentially
  • All tests not grouped into a suite will be collected in a "default" suite
